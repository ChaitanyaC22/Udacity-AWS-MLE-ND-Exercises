• Exercise: Ground Truth: Use Sagemaker Ground Truth label a sentiment dataset.

Procedures for this exercise:

• Launch the AWS Gateway.
• Upload the sentiment_data.jsonl dataset to a S3 bucket.
• Navigate to Sagemaker Ground Truth.
• Create a labeling job project using the dataset (sentiment_data.jsonl) as shown in the image below.
• Invite yourself to work on it. Use the Private worker type.
• Set the label options for the task for text single classification: positive , negative , and neutral.
• Label the dataset after receiving the private email for the task.
• Complete the labeling project by stopping the project
• Go to the output location of the labeling project in S3 and download the output.manifest file.
